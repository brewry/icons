## vue-icons
Vue icon components for Brew UI.

`vue-icons` is a high quality icon library, all components are displayed by `svg`.

<br/>

### Usage

1. install: `yarn add @brew/icons`.

2. import all:
    ```js
    import { install } from '@brew/icons'
    import Vue from 'vue'
    
    install(Vue)
    ```
